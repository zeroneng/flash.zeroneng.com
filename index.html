INSTRUCTIONS

Before you flash

  1. Connect RHYTHM to your computer using the USB port.
  2. Put RHYTHM into Flash Mode in the Settings menu.
  3. Select the firmware version you want to install.
  4. Click Flash Firmware.
  5. Approve the browser USB device prompt if it appears.
  6. Wait for the flash process to complete before unplugging the device.
  • Use a Chromium-based browser with WebUSB support.
  • Do not disconnect power or USB while flashing.
  • Only flash firmware files intended for RHYTHM.
  • If flashing fails, power-cycle the device and retry.
