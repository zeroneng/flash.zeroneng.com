Daisy Programmer

Welcome to the Daisy!

To program the Daisy, we'll need it in bootloader mode:

  1. Connect the Seed to your computer with the USB Micro cable.
  2. Press and hold the Boot button
  3. Press and hold the Reset button
  4. Let go of the Reset button
  5. Let go of the Boot button

Don't worry if it doesn't seem like anything happened, we'll find out in a second if it worked! (Windows users may need to complete a few extra steps.)

Now let's give blink a try! Go ahead -- press the button! If you don't see "DFU In FS Mode" or something similar in the pop-up menu, make sure the Daisy is plugged into the right computer and try following the above procedure again.

If all goes well, your Daisy should now be ✨ blinking ✨.

If you need to flash a binary file, head over to the "File Upload" tab. Otherwise, feel free to peruse the examples in the "Select an Example" tab.

Have fun!

README
Flash the Daisy bootloader.

The Daisy bootloader is an alternative to the built-in bootloader that allows you to flash much larger programs. The Daisy bootloader can accept new binaries during its timeout period, indicated by a breathing LED pattern. To extend this period indefinitely, press the BOOT button (without holding RESET).

For more information, visit the Daisy Docs bootloader page.
